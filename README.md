# Tic-Tac-Toe Game

A console-based Tic-Tac-Toe game written in Python.

## How to play
1. Run the script using Python: `python3 tic_tac_toe.py`
2. You will play as 'X' and the computer will play as 'O'.
3. Enter the row (0-2) and column (0-2) for your move.
4. The game will automatically check for a winner or a draw.
